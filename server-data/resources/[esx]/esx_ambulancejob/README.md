# esx_ambulancejob

ESX Ambulance Job is an plugin for ESX with features:

- Adds death screen, with early respawn timer and bleed out timer
- Vehicle garages, revive menu and more for on duty EMS

## Requirements

`Required`:

- [esx_skin]
- [esx_vehicleshop]
